# Resumen sintaxis C# vs Java

## Variables y tipos
- **Java**: `int numero = 5;`
- **C#**:  `int numero = 5;`
- **Java**: `String texto = "hola";`
- **C#**:  `string texto = "hola";`

## Condicionales
- **Java**:
  ```java
  if (cond) { }
  ```
- **C#**:
  ```csharp
  if (cond) { }
  ```

## Bucles
- **Java**:
  ```java
  for (int i = 0; i < 10; i++) { }
  ```
- **C#**:
  ```csharp
  for (int i = 0; i < 10; i++) { }
  ```

## Clases y namespaces
- **Java**:
  ```java
  package com.ejemplo;
  public class Persona { }
  ```
- **C#**:
  ```csharp
  namespace Ejemplo
  {
      public class Persona { }
  }
  ```
